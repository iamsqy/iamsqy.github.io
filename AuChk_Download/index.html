AuChk 可以帮助学校电台社团在制作广播节目前检查所使用的音频。

当前版本: V1.0 (公测版本β)

支持的操作系统: Windows 10及其他版本的 Windows。

最低系统要求

处理器: Pentium 或更高
内存: 256 MB或更高
存储空间: 16 MB用于安装应用程序
硬件: 声音卡和驱动程序
软件: Windows Media Player 已安装

推荐系统要求

处理器: Core i3 或更高
内存: 2 GB或更高
存储空间: 16 MB用于安装应用程序
硬件: 声音卡和驱动程序
软件: Windows Media Player 版本12.0.9600.17031已安装

有关下载问题

我们将发行版安装包上传到了GitHub上，同时也在Gitee上同步了GitHub的仓库。由于我们没有在Gitee上同步GitHub的仓库，Gitee上的发行版可能不是最新，因此我们建议您如果可能，优先尝试下载GitHub上的发行版，如果无法下载，请继续尝试Gitee上的发行版。在Releases界面，请选择setup.exe下载，而非源代码 (zip或tar.gz)。

由于GitHub服务器在某些地区的连接受限或被防火墙禁止，因此可能存在无法下载的情况。给出的所有链接的下载速度均较慢。如果无法下载，请与Iamsqy联系。
